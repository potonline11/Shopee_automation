1. บทนำ

ยินดีต้อนรับสู่ Shopee Shorts Poster System ("เรา", "ระบบของเรา") เราให้ความสำคัญอย่างยิ่งต่อการคุ้มครองข้อมูลส่วนบุคคลของคุณ นโยบายความเป็นส่วนตัวนี้อธิบายถึงวิธีการเก็บรวบรวม ใช้ เปิดเผย และรักษาความปลอดภัยข้อมูลของคุณเมื่อคุณเชื่อมต่อบัญชี YouTube หรือบริการของ Google ผ่านระบบของเรา

2. ข้อมูลที่เราเก็บรวบรวม

เมื่อคุณใช้งานระบบของเราเพื่อเชื่อมต่อกับ YouTube API และสร้างคอนเทนต์อัตโนมัติ เราอาจเข้าถึงข้อมูลดังต่อไปนี้โดยได้รับอนุญาตจากคุณ:

  • ข้อมูลบัญชี Google/YouTube: ข้อมูลโปรไฟล์พื้นฐาน เช่น ชื่อบัญชี อีเมล และรูปโปรไฟล์
  • การจัดการวิดีโอบน YouTube: ข้อมูลและสิทธิ์ในการอัปโหลด จัดการ หรือแก้ไขวิดีโอ Shorts ของคุณตามขอบเขตที่คุณอนุญาตผ่าน OAuth Client
  • คีย์การเชื่อมต่อ (API Keys): คีย์เชื่อมต่อที่คุณตั้งค่าไว้ในระบบ เช่น OpenAI, Leonardo AI, Runway และ ElevenLabs ซึ่งจะถูกจัดเก็บไว้เฉพาะในระดับบราวเซอร์ของผู้ใช้ (Local State) เท่านั้น

3. การใช้งานข้อมูลของคุณ

เราใช้ข้อมูลที่รวบรวมเพื่อวัตถุประสงค์ต่อไปนี้เท่านั้น:

  • เพื่อดำเนินการสร้างวิดีโอ เขียนบท และจัดเตรียมลิงก์ Affiliate ของ Shopee
  • เพื่อส่งข้อมูลและสื่อสำเร็จรูปไปยังแพลตฟอร์ม Make.com ในการอัปโหลดขึ้น YouTube Shorts โดยอัตโนมัติ
  • เราไม่มีนโยบายการขาย แลกเปลี่ยน หรือส่งต่อข้อมูลส่วนบุคคลของคุณให้กับบุคคลภายนอกที่ไม่เกี่ยวข้อง

4. ความปลอดภัยของข้อมูล

ระบบของเราไม่มีการเก็บรักษา API Keys หรือข้อมูลรหัสผ่านอันละเอียดอ่อนของคุณไว้ที่ฝั่ง Server-side อย่างถาวร ข้อมูลทั้งหมดจะถูกรักษาความปลอดภัยภายใน Web Storage (Local Storage) ของบราวเซอร์คุณเอง เพื่อป้องกันความเสี่ยงจากการรั่วไหลของข้อมูลสูงสุด

5. สิทธิ์ความเป็นส่วนตัวของคุณ

คุณมีสิทธิ์ในการเพิกถอนการเข้าถึงของแอปพลิเคชันได้ตลอดเวลา ผ่านเมนูความปลอดภัยในบัญชี Google ของคุณ (Google Account Settings -> Security -> Third-party apps with account access) ซึ่งจะตัดการเชื่อมต่อและยุติสิทธิ์การอัปโหลดของระบบในทันที
